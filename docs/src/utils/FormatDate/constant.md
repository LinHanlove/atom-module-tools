# 格式化日期为中文星期

### DATA_CONSTANT


## 时间戳

| 每次 | 说明 | 类型 | 值 |
| --- | --- | --- | --- |
| ONE_DAY | `获取1天的时间戳` | Number | - |
| ONE_HOUR | `获取1小时的时间戳` | Number | - |
| TODAY_START_TIMESTAMP | `当天的时间戳` | Number | - |
| TOMORROW | `当天 + 1天的时间戳` | Number | - |
| LAST_WEEK | `当天之前一星期的时间戳` | Number | - |
