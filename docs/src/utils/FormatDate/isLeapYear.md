# 获取某个日期是当年的第几天

### isLeapYear


## 参数

| 参数名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| date | `Number`  | - | 需要格式化的日期 |

## 返回

`boolean` 

## 代码演示

### 基础用法

```js


import { isLeapYear } from 'atom-tools'

// 传入date
console.log(2024) // true

// 传入时间戳
console.log(1714405699791) // true


```
