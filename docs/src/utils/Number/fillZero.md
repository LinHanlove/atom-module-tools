# 补零

### fillZero

## 说明
小于10的数字前面补0

## 参数

| 参数  | 说明         | 类型 | 默认值 |
| ----- | ------------ | ---- | ------ |
| num | 传入的数字 | `Number`  | ---      |

## 返回值

`String`

## 代码演示

### 基础用法

```ts
import { fillZero } from 'atom-tools';

console.log(fillZero(6)); // 06

```
