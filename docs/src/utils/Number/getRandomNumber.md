# 随机数字

### getRandomNumber

## 说明
生成随机字数字，该数字唯一

## 参数

| 参数 | 说明 | 类型   | 默认值 |
| ---- | ---- | ------ | ------ |
| digit | 自定义长度 | `Number` | 2      |

## 返回值

`string`

## 代码演示

### 基础用法

```ts
import { getRandomNumber } from 'atom-tools';

console.log(getRandomNumber(3)); // 258

```
